# Migrações Supabase

Use o [Supabase CLI](https://supabase.com/docs/guides/cli) nesta pasta quando o schema estiver definido:

```bash
supabase init
supabase link --project-ref <ref>
```

As tabelas (perfis, hubs, salas, sessões de foco, metas, etc.) ainda não foram modeladas — isso fica como próximo passo após fechar o modelo de dados.
